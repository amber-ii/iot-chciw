# IOT 電控網

## 功能
##### 物聯網 MQTT & Arduino
1. 訂閱(subscribe)、接收廠內各設備的溫度、壓力、噸數等即時資訊，以及修改、發布(publish)數值

2. 查詢廠內各設備數據歷史資料

3. Export excel

4. 即時資料、歷史資料曲線圖

5. 員工體溫異常，即時發Line Notify通知主管

## 技術
#### 前端
- JavaScript
- Tailwinds CSS
- jQuery

#### 後端
- Express.js (4.17.1)

#### 資料庫
- SQL Server(Arduino資料) (7.1.3)
- MongoDB(使用者資訊 && 設備斷線紀錄) (5.13.7)
